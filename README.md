# ebay-automation-playwright
Playwright Automation Project for ebay page using JavaScript
# Setup Instructions
01.Install Dependencies:
    Open the terminal in this folder and run
    npm -v
    node -v
    npm init playwrigt@latest

02.Run Tests:
    npx playwright test

03.View Report:
    npx playwright show-report

# Included Scenarios
01.Verifies the widget header appears.
02.Ensures no more than 6 items are displayed.
03.Parses product prices and verifies they fall within the defined range
04.Verifies the "See all" link functionality.
